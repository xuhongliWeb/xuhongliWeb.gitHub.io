## 第一篇文章 Hello hexo

### 发布流程
#### 新建
```
hexo new "hello-world"
```
#### 创建tags
```
hexo new page tags
```
参考链接 https://www.jianshu.com/p/e17711e44e00
[Hexo使用攻略-添加分类及标签](https://www.jianshu.com/p/e17711e44e00)
##### 执行命令
```
hexo g
```
#### 发布到github 
```
hexo clean
hexo g -d
```
